# Application produite avec Nest.js, Vue.js and MongoDB

API CRUD réalisé avec Nest.js, avec un front Vue.js et une base de données MongoDB.

## Débuter
Le projet est divisé en 2 sections : front et back.

### Clone the repository
Clonez le dépot pour récupérer les deux sections du projet (`app-backend` and `app-frontend`).

```bash
git clone https://github.com/AntoninJoulie/DL-Sante.git
```

## Changez de repertoire pour entrer dans le projet tout juste cloné
```bash
cd DL-Sante
```

Note à savoir, vous allez avoir besoin de 2 invite de commandes pour lancer le projet.

## Backend
### Changer de répertoire pour rentrer dans la section backend
```bash
cd app-backend
```

### Installez les dépendances du backend

```bash
npm install
```

### Lancez l'application
Dans le répertoire `app-backend` du projet, lancez la commande suivante :

```bash
npm run start
```

Le backend va se lancer sur le port `3000`.

## Frontend
Ouvrez un nouveau terminal dans `DL-Sante` et naviguez jusqu'au document `app-frontend` pour préparer le front.

### Installez les dépendances du Frontend
```bash
cd app-frontend
npm install
```

### Lancez le frontend

```bash
npm run serve
```

### Test de l'application
Enfin, vous pouvez tester l'application sur http://localhost:8080
